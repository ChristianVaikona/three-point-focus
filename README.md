# Three-Point-Focus
## Author: Christian Vaikona

As a sports photographer hobbyist, I like to delve into the nitty-gritty details of the craft every now and then. One of these details is positioning. When I show up to a game I am photographing, I wonder "Where should I be? Where might be the best spot for me to get a really cool picture?". This project aims to help answer just that.
